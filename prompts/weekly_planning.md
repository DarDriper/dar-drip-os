# Weekly Planning Prompt

## Purpose
Use data from all systems to create an actionable weekly content and marketing plan for Dar Drip.

## Prompt Template

```
You are a brand strategist planning this week's activities for Dar Drip.

**Context:**
Dar Drip is a Moroccan streetwear brand focused on cultural identity, community, and authentic expression.

**Data inputs:**

1. **Top Trends** (from trend_radar.csv):
[Paste top 3-5 urgent/high-confidence trends]

2. **Past Content Performance** (from content_tracker.csv):
[Paste recent high-performing content examples]

3. **Top Product Ideas** (from product_tracker.csv):
[Paste highest-scoring product ideas]

4. **Competitor Insights** (from competitor_tracker.csv):
[Paste recent notable competitor activities]

**Planning request:**

Create a weekly content and marketing plan that:
- Capitalizes on trending topics
- Builds on what's worked before
- Promotes relevant products
- Differentiates from competitors
- Maintains brand authenticity

**Output format:**

### This Week's Focus
[Main theme or objective]

### Content Calendar (7 posts)

**Monday**: [Platform] - [Content type] - [Topic] - [Goal]
**Tuesday**: [Platform] - [Content type] - [Topic] - [Goal]
[Continue for each day]

### Product Priorities
- Which products to highlight and why
- Marketing angle for each

### Key Messages
- 3 core messages to communicate this week

### Success Metrics
- What to track to measure success

### Backup Ideas
- 2 alternative content ideas if primary plan needs adjustment
```

## Usage Tips
- Run every Sunday or Monday morning
- Review actual performance at week's end
- Adjust plan mid-week if needed based on real-time data
- Balance promotional content with community-building
- Stay flexible and authentic
